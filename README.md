# jokenpo

Projeto de um jogo apenas para por em prática conceitos de :

 - MVC
 - Provider
 - Testes unitários 
 
# Estrutura do projeto

  - Padrão MVC
    - Funcionalidade 
      - controller
      - model
      - imagens
      - repository
      - view
   - Provider como controle de estado concentrado no AppWidget para ser utilizado em qualquer parte do app pelo o comsumer.
